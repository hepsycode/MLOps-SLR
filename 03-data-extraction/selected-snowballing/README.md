# Final Studies Added through Snowballing

This directory contains the **4 studies** identified through citation chasing and retained after full-text assessment:

- `SB01` - MetaML-Pro: Cross-Stage Design Flow Automation for Efficient Deep Learning Acceleration
- `SB13` - AutoDNNchip: An Automated DNN Chip Predictor and Builder for FPGAs and ASICs
- `SB14` - Best of Both Worlds: AutoML Co-Design of a CNN and Its Hardware Accelerator
- `SB32` - Leveraging Incremental Machine Learning for Reconfigurable Systems Modeling under Dynamic Workloads

## Role in the review

These studies complement the 22 papers selected from the automatic database search and bring the final evidence base to 26 studies. They are especially relevant to cross-stage automation, integrated model/hardware exploration, end-to-end accelerator generation, and monitoring or incremental model evolution.

Their completed coding is available in [`../final_selected_papers.csv`](../final_selected_papers.csv), where the `SBxx` identifiers connect the rows to the files in this directory and to the snowballing screening records.
