# Final Studies from the Automatic Database Search

This directory contains local full-text copies of the **22 studies** retained after full-text assessment of the 152 candidates originating from ACM Digital Library, IEEE Xplore, Scopus, and SpringerLink.

## Identifier convention

These studies use the `Pxxx` identifier family. The prefix in each filename corresponds to:

- the `paper_id` field in [`../final_selected_papers.csv`](../final_selected_papers.csv);
- the screening and extraction workbooks;
- the LaTeX citation key used in the manuscript.

## Role in the review

The studies in this directory form the primary-search portion of the final evidence base. They were coded across the RQ1, RQ2, and RQ3 dimensions defined in [`../data_extraction_form_legend.csv`](../data_extraction_form_legend.csv).

The final analytic record for each study is the corresponding row in [`../final_selected_papers.csv`](../final_selected_papers.csv), not the PDF filename or any intermediate spreadsheet row.

## Copyright notice

Before publication of the package, remove any file without an explicit redistribution license and retain the study metadata, DOI, and legal access link instead.
