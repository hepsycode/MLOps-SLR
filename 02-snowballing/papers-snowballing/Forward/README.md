# Forward Snowballing Candidates

This directory contains the **5 candidate studies** identified through forward citation searching, that is, by finding later publications that cite the seed or selected studies.

The files use identifiers `SB29` through `SB33`. Their metadata and screening decisions are documented in:

- [`../../00 - snowballing_candidates.csv`](../../00%20-%20snowballing_candidates.csv)
- [`../../04_ic_ec_compiled_final.csv`](../../04_ic_ec_compiled_final.csv)

A candidate in this directory is not automatically included in the review. Candidates that passed the initial screening were copied to [`../../selected-snowballing/`](../../selected-snowballing/) and then assessed at full text.

## Audit use

Use the `SBxx` identifier to trace a PDF to its screening record and, when applicable, to the data extraction form and final dataset.

## Copyright notice

The PDFs are local review artifacts. Before making the repository public, remove any full text that is not licensed for redistribution and preserve only the bibliographic metadata and DOI/link information.
