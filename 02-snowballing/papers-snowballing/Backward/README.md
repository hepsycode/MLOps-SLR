# Backward Snowballing Candidates

This directory contains the **28 candidate studies** identified by screening references cited by the seed and selected studies.

The files use identifiers `SB01` through `SB28`. Their bibliographic metadata and screening decisions are stored in the parent snowballing tables:

- [`../../00 - snowballing_candidates.csv`](../../00%20-%20snowballing_candidates.csv)
- [`../../04_ic_ec_compiled_final.csv`](../../04_ic_ec_compiled_final.csv)

A candidate in this directory is not necessarily part of the final review. Studies passing the initial IC/EC stage were copied to [`../../selected-snowballing/`](../../selected-snowballing/), and only four snowballing studies were ultimately included after full-text assessment.

## Audit use

Use the `SBxx` prefix to connect each PDF to its row in the snowballing screening matrix. The matrix records domain, ML, lifecycle/workflow, venue, document-type, and substantive relevance decisions.
