# Snowballing Studies Selected for Full-Text Assessment

This directory contains the **17 snowballing candidates** that passed the initial IC/EC screening and were retained for detailed full-text assessment.

This is an intermediate selection set. It should not be confused with the four snowballing studies included in the final evidence base.

## Processing relationship

- The complete 33-candidate set is stored in [`../papers-snowballing/`](../papers-snowballing/).
- Screening decisions are recorded in [`../04_ic_ec_compiled_final.csv`](../04_ic_ec_compiled_final.csv).
- The 17 selected rows are also available in [`../04_ic_ec_selected.xlsx`](../04_ic_ec_selected.xlsx).
- Full-text coding for these studies is recorded in [`../../03-data-extraction/data_extraction_form_full_snowballing.xlsx`](../../03-data-extraction/data_extraction_form_full_snowballing.xlsx).
- The four studies finally included from snowballing are stored in [`../../03-data-extraction/selected-snowballing/`](../../03-data-extraction/selected-snowballing/).

## Why some studies were later excluded

Initial screening assessed eligibility from metadata, title, and abstract. Full-text assessment then determined whether each candidate contributed sufficiently to the research questions, especially lifecycle capability, toolchain integration, and evidence useful for gap analysis. Therefore, passing this stage did not guarantee inclusion.
